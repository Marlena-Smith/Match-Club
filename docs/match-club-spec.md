# Match Club 产品规格文档

## 1. 产品概述

### 1.1 产品定位
面向大学新生的社团匹配工具，通过问卷生成用户画像，与社团标签进行智能匹配，输出个性化推荐结果。

### 1.2 核心机制
- 学生填写问卷，系统生成兴趣标签和偏好画像
- 社团填写后台问卷，系统提取匹配特征
- 基于多维度字段进行加权匹配，输出匹配度百分比
- 学生可浏览、收藏、申请加入社团

### 1.3 目标用户
- **学生端**：大学新生，希望快速找到适合自己的社团
- **社团端**：社团组织者，希望高效招募匹配的新成员

---

## 2. 用户流程

### 2.1 学生端流程
```
Landing Page（选择身份：学生）
    ↓
首页（登录弹窗）
    ↓
填写问卷（生成画像标签）
    ↓
查看匹配结果（显示匹配度百分比）
    ↓
浏览社团详情（社团宣传海报）
    ↓
提交入社申请
    ↓
等待消息通知（同意/拒绝）
```

### 2.2 社团端流程
```
Landing Page（选择身份：社团组织）
    ↓
首页（登录弹窗）
    ↓
填写社团问卷（后台匹配数据，不展示给学生）
    ↓
制作社团宣传海报（展示给学生）
    ↓
预览效果
    ↓
查收入社申请消息
    ↓
同意/拒绝申请
```

## 3. 页面结构

### 3.1 Landing Page（共用）
- 产品介绍
- 身份选择入口：学生 / 社团组织

### 3.2 学生端页面（9个）

| 序号 | 页面名称 | 功能描述 |
|------|----------|----------|
| 1 | 首页 | 登录入口、补充信息入口、匹配社团窗口 |
| 2 | 问卷填写页 | 学生问卷，有交互感的表单设计 |
| 3 | 匹配结果页 | 显示匹配度百分比、社团标签列表 |
| 4 | 社团信息页 | 查看社团宣传海报详情 |
| 5 | 申请入社页 | 自动填充信息 + 自荐内容输入 |
| 6 | 浏览页 | 滑动浏览所有社团（类似探探） |
| 7 | 收藏列表页 | 已收藏的社团列表 |
| 8 | 消息页 | 申请结果通知（同意/拒绝） |
| 9 | Profile页 | 编辑偏好、查看标签、已加入社团 |

**底部导航栏**：首页、浏览、收藏、Profile（4个Tab）

### 3.3 社团端页面（6个）

| 序号 | 页面名称 | 功能描述 |
|------|----------|----------|
| 1 | 首页 | 登录入口、社团信息填写入口 |
| 2 | 问卷填写页 | 社团后台问卷（用于匹配，不展示） |
| 3 | 海报编辑页 | 制作社团宣传海报 |
| 4 | 海报预览页 | 复用学生端社团信息页 |
| 5 | 消息页 | 查收入社申请、同意/拒绝操作 |
| 6 | 新人页 | 已同意加入的学生列表（昵称、标签、大头贴、自荐内容） |

**底部导航栏**：首页、问卷、海报、新人（4个Tab）

**新人页说明**：
- 展示所有在"消息页"中被社团同意的学生用户
- 列表形式展示每位新人的：昵称、大头贴、兴趣标签、自荐内容
- 方便社团管理员查看和管理已录取的新成员

## 4. 数据结构

### 4.1 学生问卷字段（student_info）

| 字段名 | 字段类型 | 说明 | 匹配优先级 | 前端形式 | 匹配指向 |
|--------|----------|------|------------|----------|----------|
| student_id | 文本/数字 | 学生唯一标识（主键） | 不参与匹配 | 系统生成 | - |
| name | 文本 | 姓名 | 不参与匹配 | 输入框 | - |
| nickname | 文本 | 昵称 | 不参与匹配 | 输入框 | - |
| gender | 文本 | 性别：男/女/其他 | 不参与匹配 | 下拉框 | - |
| avatar | 图像 | 大头贴（图片路径/链接） | 不参与匹配 | 图片上传 | - |
| interest_tag_primary | 文本 | 兴趣一级标签：体育/艺术/学术/技术/娱乐/公益/社交 | 中级 | 下拉菜单/单选 | tag_primary |
| interest_tag_secondary | 文本 | 兴趣次级标签（一级标签下细分） | 中级 | 下拉菜单/单选 | tag_secondary |
| interest_tag_custom | 文本 | 自定义兴趣标签 | 不参与匹配 | 输入框 | - |
| experience_level | 文本 | 经验程度：想尝试/已玩过/老玩家 | 高级 | 下拉菜单/单选 | recruitment_threshold |
| prefer_activity_frequency | 文本 | 可接受活动频率：高频/每周/偶尔 | 中级 | 下拉菜单/单选 | activity_frequency |
| prefer_weekly_hours | 文本 | 每周可投入时间：<2h/2-5h/5h+ | 中级 | 下拉菜单/单选 | weekly_hours |
| max_fee_acceptable | 数字 | 可接受最高会费：0/10/20/30/50/60/80/100 | 高级 | 下拉栏 | membership_fee |
| need_newbie_training | 文本 | 是否需要新手村培训：是/否 | 高级 | 下拉菜单/单选 | has_newbie_training |
| participation_philosophy | 文本 | 参与理念：想要自由/想要监督 | 高级 | 下拉菜单/单选 | club_philosophy |
| social_tendency | 文本 | 社交倾向：我带氛围/开团秒跟/社恐求带 → 映射为外向/中间/内向 | 低级 | 下拉菜单/单选 | social_expectation |
| group_preference | 文本 | 群体偏好：想要大家庭/想要小团体 | 中级 | 下拉菜单/单选 | group_type |
| main_goal_1 | 文本 | 主要目标（优先级1） | 中级 | 多选排序 | growth_value |
| main_goal_2 | 文本 | 主要目标（优先级2） | 中级 | 多选排序 | growth_value |
| main_goal_3 | 文本 | 主要目标（优先级3） | 中级 | 多选排序 | growth_value |

**学生问卷前端分组**：
- 基础信息：姓名、昵称、性别、大头贴、兴趣标签
- 投入与精力：活动频率、每周投入、社费额度
- 性格与社交偏好：参与理念、社交倾向、群体偏好
- 目标导向：主要目标（按优先级排序）

- 成长路径：本社能带来的价值（多选）

## 5. 设计规范

### 5.1 色彩
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #F5B70A | 明黄，主要按钮、强调元素 |
| 次色 | #0095B6 | 青蓝，次要按钮、链接 |
| 点缀 | #AE322A | 红棕，警示、删除、点缀 |
| 背景白 | #F9F6E5 或 #FCEFE8 | 温暖的米白色背景 |

### 5.2 字体
- **字体家族**：苹方（PingFang SC）
- **字号规范**：11 / 13 / 16 / 20 / 24 / 28 / 36 / 48 px

### 5.3 间距
- **间距规范**：4 / 8 / 16 / 24 / 36 / 48 px

### 5.4 图标尺寸
- **图标规范**：20 / 24 / 36 / 48 px

### 5.5 画框尺寸
- **设计稿尺寸**：390 x 844 px（移动端优先）
- **左右边距**：16 px

### 5.6 视觉风格
- 线条风 + 卡通风
- 年轻活泼，适合大学生群体
- 圆角卡片，柔和阴影

## 6. 特殊状态

### 6.1 空状态设计
- **无匹配结果**：显示鼓励文案 + 重新填写问卷入口
- **无消息**：显示"暂无消息"插画
- **无收藏**：显示"快去发现社团吧"引导

### 6.2 加载状态设计
- **匹配计算中**：显示有趣的加载动画（如转动的齿轮、飞舞的标签）
- **数据加载**：骨架屏 + 品牌色加载条

### 6.3 错误状态
- 网络错误提示
- 表单验证错误提示

## 7. 匹配算法逻辑（参考）

### 7.1 匹配优先级权重
| 优先级 | 权重 | 字段示例 |
|--------|------|----------|
| 高级 | 3 | experience_level, max_fee_acceptable, need_newbie_training, participation_philosophy |
| 中级 | 2 | interest_tag_primary/secondary, activity_frequency, weekly_hours, group_preference, main_goal |
| 低级 | 1 | social_tendency |

### 7.2 匹配规则
- **精确匹配**：标签完全一致得满分
- **范围匹配**：如会费，学生max >= 社团fee 得满分
- **多选匹配**：如growth_value，计算交集比例
